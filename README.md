# RAVENSPIRE

See every chain. Fear no rug. Rule your realm.

An AI-native, creator-first social and on-chain intelligence platform for serious
Ethereum and DeFi investors, wrapped in the world of an epic medieval-fantasy saga.

## Stack

- Next.js (App Router) + TypeScript + Tailwind CSS + Framer Motion
- Supabase (database, auth, realtime, storage) with row-level security
- viem + wagmi, non-custodial embedded wallets (ERC-4337 ready)
- Anthropic-powered AI (The Raven)

## Getting started

```bash
npm install
cp .env.example .env.local   # fill in keys
npm run dev
```

## Scripts

- `npm run dev` starts the dev server
- `npm run typecheck` runs the TypeScript compiler
- `npm run build` builds for production
- `npm run lint` runs ESLint

## Design system

"Obsidian & Gold". Dark glass surfaces, gold hairlines, bone text, warm
candlelight accents. Display type is Cinzel, UI type is Inter with tabular
numerals. Tokens live in `app/globals.css`.
